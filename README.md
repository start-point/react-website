# umi project

## Getting Started

🎉安装依赖

(注: 如果安装后 运行 出现错误 可能就是包依赖没有下载完全 你可以通过npm i 去安装一下 )

```bash
$ yarn
```

🎉项目启动

```bash
$ yarn start
```

## 项目结构

```s
|-- config      -- 配置
    |-- config     -- 路由配置
    |-- routes   -- 路由
|-- public      -- 依赖的静态资源存放
|-- src
    |-- pages       -- 页面
    |-- utils       -- 工具类
    |-- app.tsx     -- 入口
    |-- global.less -- 全局样式
```

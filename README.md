# umi project

## Getting Started

🎉安装依赖

```bash
$ yarn
```

🎉项目启动

```bash
$ yarn start
```

## 项目结构

```s
`-- config      -- 配置
    |-- config     -- 路由配置
    |-- routes   -- 路由
|-- public      -- 依赖的静态资源存放
`-- src
    |-- pages       -- 页面
    |-- utils       -- 工具类
    |-- app.tsx     -- 入口
    |-- global.less -- 全局样式
```
